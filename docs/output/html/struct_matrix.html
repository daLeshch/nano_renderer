Detailed Description

template<size_t rows, size_t cols, typename T = double>
struct Matrix< rows, cols, T >

Generic matrix template for mathematical and graphics applications.

Supports addition, subtraction, scalar and matrix multiplication, determinant, transpose, and inversion operations.

Template Parameters
rowsNumber of rows.
colsNumber of columns.
TData type (default double).

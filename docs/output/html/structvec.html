Detailed Description

template<typename T, int N>
struct vec< T, N >

Generic N-dimensional vector template.

This structure supports mathematical operations such as addition, subtraction, dot and cross products, normalization, and scalar operations.

Template Parameters
TData type of elements (e.g., float, double, int).
NNumber of elements in the vector.

***Новое:

Добавлена структура для работы с матрицами. Реализованы такие методы как поиск детерминанта матрицы,
инверсия матрицы, транспонирование. Переопределены операторы для работы со скалярами, матрицами и векторами.
Добавила тесты для модуля с матрицами и векторами, чтобы дополнительно проверить их логику.

Основная математическая часть работы завершена.
***


Основная задача проекта -- разобраться в работе openGL в процессе написания своей урезанной версии не используя никаких сторонних графических библиотек.
Базовая рабочая версия уже есть, нужна доработка.

Что уже реализовано в первой версии проекта:

- Считывание и разбор на точки obj файла
- Визуализация линий
- Построение bounding box
- Визуализация заполненных треугольников
- Простая реализация backface culling
- Работа с библиотекой записи в TGA(костыль)

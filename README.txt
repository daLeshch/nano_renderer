Основная задача проекта -- разобраться в работе openGL в процессе написания своей урезанной версии не используя никаких сторонних графических библиотек.
Базовая рабочая версия уже есть, нужна доработка.

Что реализовано:

- Считывание и разбор на точки obj файла
- Визуализация линий
- Построение bounding box
- Визуализация заполненных треугольников
- Простая реализация backface culling
- Работа с библиотекой записи в TGA(костыль)

 Пример вывода нынешней реализации приложен, файл "example.tga"

Что доделать:

- Свою реализацию записи в файл(BMP или TGA)
- Изменить алгоритм backface culling(сейчас неправильно обрабатывает завернувшиеся внутрь полигоны)
- Добавить перспективную камеру
- Добавить подгон кадра под размер модели ИЛИ реализовать вьюпорт(если успею)
- Реализовать простенький шейдер
- Попробовать разбить процесс рисования на потоки(проверить будет ли это быстрее)


Для запуска не обязательно передавать путь к файлу, по умолчанию поймает сам.
Можно передать до 3х аргументов: путь к файлу, ширина и высота картинки.
